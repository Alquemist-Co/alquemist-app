Detalles del diseño

📊 Sección Global (arriba)

  • 5 KPIs: Activos, Transición, Espera, Completados, Duración prom.
  • Clickeables → navegan a la lista filtrada
  • Colores semánticos (verde/amarillo/rojo)

🔧 Toolbar

  • Dropdown tipo de cultivo (cambia las fases visibles)
  • Botón filtros: instalación, estado, cultivar
  • Resumen contextual a la derecha

📋 Phase Stepper

  • Línea vertical tipo stepper conectando fases
  • Cada fase = card con analytics en header
  • Lotes colapsables (2 visibles + "ver más")
  • Colores por fase del modelo de datos

🏷️ Fila de lote

  • Dot de estado (verde=ok, amarillo=transición, rojo=espera/vencido)
  • Código mono + cultivar + badge de estado si no es "activo"
  • Plantas, zona, días/esperados, mini barra de progreso
  • Click → navega a detalle del lote
